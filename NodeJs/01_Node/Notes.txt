Node.js, Google Chrome ke V8 JavaScript Engine ka use karta hai.

🔹 V8 Engine kya hai?

V8 ek JavaScript engine hai jo Google ne banaya hai aur ye Google Chrome browser ke andar JavaScript run karta hai.

Node.js bhi isi V8 engine ko use karta hai JavaScript code ko machine code me convert karne ke liye.

🛠 Simple Flow Samjho:

JavaScript Code → V8 Engine → Machine Code → Run on Computer

Bas difference ye hai:

Chrome me V8 browser ke andar run hota hai.

Node.js me V8 system ke andar standalone run hota hai (server side).
